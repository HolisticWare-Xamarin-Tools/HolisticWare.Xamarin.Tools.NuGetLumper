# HolisticWare.Xamarin.Tools.NuGetLumper

HolisticWare.Xamarin.Tools.NuGetLumper

```
nuget => nugget => lump
```

Small utility for NuGet search (existence) with version and dependency analysis utilizing NuGet Client API.

Wrapper around APIs in:

*   `NuGet.Protocol`

*   `NuGet.Resolver`
